# Fureverly

Fureverly is a **comprehensive pet product application system** that allows users and admins to manage products, handle orders, and export order details. It comes with a modern UI, theme toggle, and PDF download functionality for orders.

---

## Features

- **Theme Toggle**: Switch between light and dark modes seamlessly.  
- **Product Management**: Add, update, and delete products.  
- **Order Management**: Place orders and download order details as PDF.  
- **Responsive Design**: Works on all screen sizes.  
- **Smooth Animations**: Enhanced UI with Framer Motion.  
- **Notifications & Alerts**: Using React Hot Toast and SweetAlert2.

---

## Tech Stack / Libraries

- **React** & **React DOM** – Frontend framework  
- **Firebase** – Backend and database  
- **React Router** – Navigation and routing  
- **TailwindCSS** – Styling  
- **Framer Motion** – Animations  
- **Lucide React** & **React Icons** – Icons  
- **React Hot Toast** & **SweetAlert2** – Notifications and alerts  
- **React Simple Typewriter** – Typewriter text effects  
- **React Spinners** – Loading spinners  
- **React Tooltip** – Tooltips  
- **jsPDF** & **jsPDF AutoTable** – PDF generation
